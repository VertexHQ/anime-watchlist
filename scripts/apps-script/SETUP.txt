Google Apps Script API setup

1) Open the Google Sheet that will store anime rows for each user.
2) In the sheet, open Extensions > Apps Script.
3) Replace the default script content with scripts\apps-script\Code.gs from this repo.
4) Save the script project.
5) Run `setupProperties()` once from the Apps Script editor to store script properties.
6) Deploy as a web app:
   - Deploy > New deployment
   - Type: Web app
   - Execute as: Me
   - Who has access: Anyone
7) Copy the web app URL and use it as `VITE_SHEETS_SCRIPT_URL`.

Required status tabs (auto-created if missing):
- watching
- completed
- plan
- coming-soon
- dropped

Anime row headers are managed automatically from `APP_CONFIG.FIELDS`, including:
- id, title, status, rating, notes, image
- droppedReason
- releaseDate
- releaseDateNotDeclared
- createdAt

Status normalization supports legacy values, including:
- upcoming -> coming-soon
- droped -> dropped
